CH 12 Canvas, SVG, and Drag and Drop

Draggable elements

In order to make elements draggable, we can add the draggable attribute to them and set the value to true


Note: draggable is not a boolean, thus, it must be explicitly set to true


The Drag and Drop API

1.) set the draggable attribute on any HTML element you want to drag


2.) add the dragstart event on the draggable html element


3.) add an event listener for the dragover and drop events
