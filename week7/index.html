CH 11. Further Functions


Async Functions

The async keyword can make a function async. by declaring the function like so: "async function()"


The keyword "await" can be used to ensure the next line of code in an async function is not executed until finished


Cool Javascript Function Facts

Functions are first-class objects in JavaScript.


Functions can be given properties by initializing them using dot notation. Ex: square.description = “ “


Immediately Invoked Function Expression

IFFE (Immediately Invoked Function Expressions) can be created by placing () around functions in JS.


IFEEs allow us to invoke the function as soon as its defined. IFFEs create variables that disappear after finished
