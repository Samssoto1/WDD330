Client-side form validation


Novalidate Form Attribute

The novalidate form attribute can be used to turn off the browser's automatic validation. THis doesn't, however, disable validation API of it is being used.


The Required Attribute

The required attribute can be used to make an input necessary to be filled out before form submission.


The Constraint Validation API

This API can be used to implement easy to use methods for form validation.


Using Fetch


Fetch allows us to work with APIs easily


The Syntax is as follows: fetch("link");


With Fetch, not only are we able to work with APIs, but we can also work with them while making promises.


We can use a .then(response => response.json()) to return the API data as JSON once it's approved.


We can specify a function to be async or not when working with APIs if we prefer to do it that way.
